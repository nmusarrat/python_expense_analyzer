# python_expense_analyzer
This project analyzes personal expenses using Python.

##Features
1.Reads expense data from a CSV file
2.Calculates total,avarage and largest expenses
3.Groups spending by category
4.Sorts categories by total spending
5.Filters expenses
6.Exports a summary to Excel
7.Generates charts of spending categories

##Technologies used
1.python
2.pandas
3.matplotlib

##Example usage
run the script:expense_analyzer.py
the program analyzes 'expense.csv' and creates an Excel report
